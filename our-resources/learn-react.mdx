# Learn React from Zero to Professional

React is the most popular frontend library for building modern user interfaces.

## Use Hooks
React hooks like `useState` and `useEffect` make life easier.

```tsx
import { useState } from 'react';

function Counter() {
  const [count, setCount] = useState(0);
  return <button onClick={() => setCount(count + 1)}>{count}</button>;
}
```
