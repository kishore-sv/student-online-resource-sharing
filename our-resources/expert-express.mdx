# Become an Express Expert

Express is the standard web framework for Node.js.

## Middleware
Build powerful pipelines for your requests.

```javascript
const express = require('express');
const app = express();

app.use((req, res, next) => {
  console.log(`${req.method} ${req.url}`);
  next();
});

app.get('/', (req, res) => res.send('Expert Express!'));
app.listen(3000);
```
